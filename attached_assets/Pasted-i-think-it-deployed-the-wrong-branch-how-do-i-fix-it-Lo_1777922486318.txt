Looking at the screenshot, Render is deployed from main but all our work is on claude/vc-labs-research-xx9j0. Two options:

Option 1 — Change Render to deploy from your feature branch (quickest)

In Render dashboard → click Settings
Scroll to Branch under "Build & Deploy"
Change main to claude/vc-labs-research-xx9j0
Click Save → then hit Manual Deploy
Option 2 — Merge the branch into main

In your terminal:

cd ~/path/to/dudilig
git checkout main
git merge claude/vc-labs-research-xx9j0
git push origin main
Render will auto-deploy on the push.

I'd go with Option 2 — merging into main is the cleaner long-term setup. Option 1 works for a demo but means your production branch isn't main, which gets confusing.
